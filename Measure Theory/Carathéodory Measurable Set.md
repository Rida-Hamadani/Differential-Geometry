### Types
- `X` : Set
- `M` : $P\left( X \right)$
- $\mu^* : P(X) \to \left[ 0,+\infty \right]$ is an [[Outer Measure]]
### Definition
$M$ is *Carath√©odory measurable* if for every $S \in P(X)$, the following is true:
$$
\mu^*\left( S \right) = \mu^*\left( S\cap M \right) + \mu^* \left( S\cap M^c \right) 
$$
##### Notation
- `M` is $\mu^*$-measurable

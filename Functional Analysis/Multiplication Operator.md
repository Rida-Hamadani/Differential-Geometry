#definition
### Types
- $\mathbb{K}$ : [[Field]]
- `V` : [[Vector Space]]
- `a` : $\mathbb{K}-\left\{ 0 \right\}$
### Definition
The *multiplication by `a` operator* is the mapping $\tau_{a} : V \to V$ defined as $\forall x\in V:\tau_{a} \left(x  \right)=a\cdot x$. 
##### aka
- Multiplication mapping
